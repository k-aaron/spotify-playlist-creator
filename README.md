# spotify-playlist-creator
Spotify 플레이리스트 자동 생성 웹앱 — 곡 목록으로 플레이리스트 생성, 기존 플레이리스트 곡 추가 및 중복 제거


## [사용법] 앱 접속 & 첫 설정
1. https://k-aaron.github.io/spotify-playlist-creator/ 접속
2. Client ID / Secret 입력
3. Redirect URI에 위 GitHub Pages URL 입력
4. Connect with Spotify → 완료


## Spotify Developer Dashboard 설정
1. [developer.spotify.com/dashboard](https://developer.spotify.com/dashboard) 접속
2. 앱 선택 → Settings
3. Redirect URIs에 아래 추가:
   https://your-username.github.io/spotify-playlist-creator/
4. Save
